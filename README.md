# Palavriando

Jogo simples de adivinhar a palavra desenvolvido com React, Vite, TypeScript e CSS. Este projeto foi criado com o objetivo de praticar e aplicar conceitos fundamentais de desenvolvimento front-end moderno utilizando o ecossistema React.

## Demonstração

Acesse a versão online do jogo:  
[palavriando-project.vercel.app](https://palavriando-project.vercel.app/)

## Tecnologias utilizadas

- Vite — Build tool para front-end rápido e moderno
- React — Biblioteca JavaScript para criação de interfaces de usuário
- TypeScript — Superset do JavaScript tipado para maior segurança no código
- CSS — Estilização visual da interface

## Como executar localmente

1. Clone o repositório:

```git clone https://github.com/DeveloperKairo/palavriando-project.git```

2. Navegue até o diretório do projeto:

```cd palavriando-project```

3. Instale as depedências:

```npm install```

4. Execute o projeto em modo de desenvolvimento:

```npm run dev```

5. Acesse em seu navegador: `http://localhost:5173` (ou conforme instruções do terminal)

## Objetivo do Jogo

O objetivo do Palavriando é adivinhar corretamente a palavra secreta. O jogo foi projetado como uma ferramenta de aprendizado e demonstração de componentes React, manipulação de estado e integração de TypeScript em um projeto prático.

## Estrutura do Projeto

- `src/`: Contém todos os arquivos fonte do projeto.
- `public/`: Arquivos estáticos.
- `package.json`: Dependências e scripts do projeto.
- `vite.config.ts`: Configuração do Vite.
